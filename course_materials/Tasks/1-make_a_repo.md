# Making a repository on GitHub

1. Navigate to your GitHub home page --> Repositories (https://github.com/orgs/Norfolk-Suffolk-ICS/repositories), make sure you are signed in

2. Click the green color 'New Repository' button on top right

3. Give your repository a name in the box provided, avoiding spaces or special character other than underscores or hyphens. If unsure you can use 'testrepo' or similar for now. Make sure that the owner is your own account (i.e. your GitHub username, and not an organisational account)

4. You can either use a pre defined template for 'Python' OR 'R' from the dropdown under 'Start with a template' OR Leave all other options as is, and click 'create repository' to make it from scatch. 

5. In the 'Choose visibility *' option, you should keep repositories private by default to minimise the risk of disclosing data.

6. And that's it, you've made your first repository!
